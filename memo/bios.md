# BIOS機能

## 文字表示

https://wiki.osdev.org/BIOS

AHレジスタとINTオペコードの2つを使って、BIOSの機能を呼び出すことができる。
たとえばAH = 0x0e INT = 0x10 をセットすると、ALレジスタにある文字を1文字表示する。

```
  mov ah, 0x0e
  mov al, 'A'
  int 0x10
```

## ブートセクタのお決まり処理

BIOSブートセクタは512バイトで、最後の2バイトに0x55AAが必要。

```
times 510 - ($ - $$) db 0
dw 0xaa55
```

NASMでは`$`は現在のアドレス、`$$`は現在のセクションの先頭アドレスである。だから、`$ - $$`は「ここまでに何バイト出力したか」となる。
たとえば100バイト使っていたら、`times 510 - 100 db 0`となり、つまり 410バイト分0を出力する。

`dw 0xaa55` はブートセクタ署名を表す。BIOSはディスクの先頭512バイトを読み込み、その最後の2バイトが`55 AA` であることを確認する。
ただし、リトリエンディアンなので実際には `55 AA`が出力される。
